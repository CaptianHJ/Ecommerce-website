🛒 E-Commerce Website
📌 Project Overview

This project is a full-stack E-commerce web application developed to simulate an online shopping platform.
Users can browse products, add items to the cart, and manage their accounts.
The project demonstrates backend development, database integration, and REST API implementation.

🚀 Features
User Registration and Login
Product Listing
Add to Cart functionality
Product Management
Database integration
REST API for product and user operations
